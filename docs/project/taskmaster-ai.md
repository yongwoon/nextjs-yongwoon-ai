# Taskmaster AI

## Getting start

- 프로젝트에서 Taskmaster AI를 사용하려면 먼저 초기화가 필요합니다. 터미널에서 `task-master init` 명령어로 Task Master를 초기화하면, Cursor 모델에게 개발 방법을 알려주는 커스텀 Cursor 규칙이 생성됩니다.

## Command

- Show task list

```bash
task-master list
```
